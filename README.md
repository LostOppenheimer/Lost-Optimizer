WINDOWS

  Download the raw "LostOptimizer" file (or copy its code), and save it as a .bat file.
  Run as administrator.
  The script will execute sequentially.
  For best results, run periodically or after installing software.

ARCH LINUX

  Copy the code into a text editor and save as LostOptimizer.sh (or any name, just put ".sh" at the end.
  CD into the folder you put the .sh file. (eg. cd Downloads)
  Make it executable: chmod +x [your file name].sh.
  Run as root: sudo ./[your file name].sh.
  The script will execute sequentially.
